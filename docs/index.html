Ad Hoc GitHub Reviewer

A public read-only view of the review workflow. The live operator app runs on Node.js so it can queue background reviews, clone repositories, inspect logs, prepare proposals, and push approved changes.

What someone can check here

  • The app purpose and user journey.
  • The review output model: errors, fixes, proposals, and approval status.
  • How the operator console is expected to be run locally or on a Node host.

What needs the Node app

  • Cloning a public GitHub repository.
  • Running repository checks and collecting logs.
  • Approving or rejecting proposals, creating commits, and pushing approved records.

Run the full app

git clone https://github.com/daljeetkohli-sap/BugfixGitrepo
cd BugfixGitrepo
npm start

Then open http://localhost:4173 for the operator console or http://localhost:4173/viewer.html for the live read-only viewer.
